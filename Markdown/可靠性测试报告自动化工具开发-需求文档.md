# 可靠性测试报告自动化工具开发-需求文档

​	请帮我完成一个本地离线网站（打包发布以后不连接网络也可使用），用于规范可靠性测试结果的输出格式，自动生成可靠性测试报告PDF文档。要求网站UI美观。

​	网页顶部设置4个标签：报告编辑（3个子标签：通用模板、通用（英文）模板、华为模板）、模板编辑（3个子标签：通用模板、通用（英文）模板、华为模板）、关于、更新日志。

​	尽管在标签栏中“报告编辑”位于“模板编辑”之前，但是“报告编辑”中的“通用模板”的内容要确保是基于“模板编辑”中的“通用模板”的内容，在编程实现时，应该是先完成“模板编辑”中的“通用模板”的功能开发。

​	顶部的“**深圳市欣威智能有限公司**”，以及底部的“测试员：”、“审核：”、“核准：”、“备注：保存部门：使用部门 保存年限：一年  [ ]绝密  [ ]机密  [ ] 内部公开  [ ] 外部公开”内容，在网页模板中所处位置与“通用.xlsx”一致

​	“通用.xlsx”文件中字体为蓝色的文字部分以及用户使用中需频繁修改的内容，需要在“报告编辑”中被设置为可编辑的文本内容（是通过Excel中的蓝色字体来识别应该被设置为可编辑内容的区域，而不是可编辑内容字体要设置为蓝色）。Excel中的黑色字体的文字部分（即日常使用中通常固定不变的模板），在“报告编辑”中为固定内容，需要在页面中展示出来，但不可编辑；而在“模板编辑”中则是可编辑内容，用于修改模板。在开发网页时要求美观且方便后续功能实现，可适当修改以实现更美观的排版和重点内容的突出显示等等实用的功能。

1.1、通用模板

​	可编辑的文本内容需要能够修改颜色（默认为黑色），字体（默认为微软雅黑），字号（默认为11号），这些设置放置在页面上方的设置区域并实现粘性定位。需要有一键框选多个文本区域的功能，以便一次性修改多个文本的字体设置。

​	每个区域的高度（或位置）可根据文本内容进行自适应调整，同时也能允许手动调整，如果因为内容过多导致一个区域被分割为两页，应在第二页起始处也有该区域的标题。比如“测试图片”区域，如果分为了两页，每页都应该有“测试图片”这一标题。

​	“**测试结果信息**”区域中，默认为3行，可以自行删减和添加。

​	“**测试图片**”区域中，默认为3行，可以自行删减和添加。每行有3个方框，对应“测试前、测试中、测试后”三种情况，每个方框内需要能够上传单张或多张图片文件，既能够通过选择文件或者文件夹上传，也可以拖拽文件或者文件夹上传。方框的宽度通过此行宽度除以3来确定，高度则与宽度一致。方框内的图片支持拖动调整顺序，并且能根据图片的数量和高宽自适应调整图片在方框内布局，确保图片在等比例的基础上能够自动缩放到尽量填充满方框区域。比如有一张正方形的图片，上传后就应该填充整个方框，如果有两张宽度是高度两倍的图片，上传后就应该自适应调整为纵向依次排列，保持等比例的基础上将宽度缩放到填充满方框的宽度。

​	在网页页面中需要能够直接看出来哪些内容属于哪一页PDF，确保导出的PDF中各部分内容所在位置和网页上显示的一致

2、模板编辑

​	点击“模板编辑”后，会有三个子标签，通用模板、通用（英文）模板、华为模板，用于修改内容填写界面中无法编辑的内容。

2.1、通用模板

​	在“模板编辑”标签页中的“通用模板”中，整个模板展示为一页即可。但是这些内容在“报告编辑”标签页中的“通用模板”中展示出来时，需要符合真实PDF的情况，包括但不限于要注意：顶部标题等内容、底部备注内容需要在每一页都具备，具体页数根据中间部分内容所占据高度自适应调整。

​	顶部的“**深圳市欣威智能有限公司**”，以及底部的“测试员：”、“审核：”、“核准：”、“备注：保存部门：使用部门 保存年限：一年  [ ]绝密  [ ]机密  [ ] 内部公开  [ ] 外部公开”内容，尽管是在“模板编辑”页面中进行内容修改，但是注意这些内容需要在“报告编辑”中的“通用模板”中的每一页都显示。“[ ]绝密  [ ]机密  [ ] 内部公开  [ ] 外部公开”内容中需要设置一个打√的功能，并且只能选中某一项进行打勾。

​	可编辑内容区域和固定文本内容在模板编辑界面中均可编辑，固定文本内容需要能够修改颜色（默认为黑色），字体（默认为微软雅黑），字号（默认为11号），这些设置放置在页面上方的设置区域并实现粘性定位。。可编辑内容区域在模板编辑界面中也可编辑，该内容作为报告编辑界面中的“填写提示”，颜色默认为灰色，字体默认为微软雅黑，字号默认为11号。需要有一键框选多个文本区域的功能，以便一次性修改多个文本的字体设置。

​	左上角需要有一个位置能够上传图片作为logo，需要能够调整大小和位置。底部的“测试员：廖燕”、“审核：冯益”、“核准：马凌锋”中，“廖燕、冯益、马凌锋”这三个名称需要更换为电子签名，因此需要能够在“测试员：”、“审核：”、“核准：”的右侧具备上传图片的功能，需要能调整大小，位置则固定为对应文字的右侧。此外，还需要一个可以自由拖拽移动的上传图片的位置，我会上传一个透明背景的部门章的图片，需要可自由调整大小和位置，且该图片在“报告编辑”界面中需增加透明度或者提供一个选项让其暂不显示，以免因遮挡影响正常编辑，但在导出为PDF时需要正常显示。这些内容的设置界面在左侧侧边栏中进行实现，在页面中仅需要实现logo图片和部门章图片位置的拖拽功能。上述图片调整大小时都是固定长宽比进行调整

​	能够导出或者加载用户设置的模板，以便不同的用户分享自己设置的模板

3、关于

​	需要有该工具的使用说明，其次需要有版本（V1.0）、开发人员姓名（邓景华）以及开发日期（年月日）。

4、更新日志
	用于查看工具的版本更新历史和详细更新内容。更新日志界面中，显示一条条记录（只包含版本、简单标题和时间），点击后会跳转到详细内容界面，随后可以点击“返回”控件返回到“更新日志”界面。
